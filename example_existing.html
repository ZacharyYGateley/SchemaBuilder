SchemaBuilder
SchemaBuilder is a simple JavaScript library that creates independent floating panels that can be dragged and dropped with the mouse.

In this example, the element already existed. It was built into the DOM on page load. SchemaBuilder popped it from the DOM and reinserted it as a position absolute element in the top element.

This panel was instantiated with the following code. 

	  var mySchema;
	  function myInit() {
		  var de = document.documentElement;
		  var bdy = document.getElementsByTagName('body')[0] || document.body || de;

		  // Create schema
		  var schema = {
			  schema: {
				elementID: 'pre-panel',
				metrics: { x: 400, y:500 }
			  }
		  };

		  // Create schema panel
		  bdy.style.height = (bdy.innerHeight || de.clientHeight)+'px';
		  mySchema = new SchemaBuilder({
			element: bdy,
			schema: schema
		  });
	  }
